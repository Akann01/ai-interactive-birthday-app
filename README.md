# 🎂 AI Interactive Birthday App

An AI-powered interactive birthday experience that runs entirely in the browser.  
Light a virtual birthday cake with your hands and blow out the candles using real-time webcam hand tracking.

Built with JavaScript and MediaPipe Hands for gesture-based interaction.

---

## ✨ Features

- 🎂 Interactive virtual birthday cake
- ✋ Real-time hand tracking using webcam
- 🔥 Light candles with hand movement
- 💨 Blow out candles using gestures
- 🎵 Automatic birthday music playback
- ⚡ Runs completely in-browser
- 🔒 Fully client-side (no backend required)

---

## 🧠 How It Works

The application uses **MediaPipe Hands** to detect hand landmarks through your webcam in real time.

### Interactions

- Move your hand near the virtual match to ignite the candles
- Perform a blowing gesture to extinguish the flames
- Enjoy the birthday celebration with music and animations

Everything is processed locally in your browser — no installation or server required.
